# Security Policy

## Supported Version

This project is currently maintained as a single active line:

- `1.x`

## Reporting a vulnerability

If you discover a security issue, please do not open a public issue with exploit details.

Instead, report privately to the repository owner with:

- A clear description of the issue
- Reproduction steps
- Potential impact
- Suggested remediation (if available)

The maintainer will acknowledge receipt and provide follow-up steps.

## Data handling

- This is a frontend-only game.
- Leaderboard entries are stored locally in browser `localStorage`.
- No game data is transmitted to external services by default.
